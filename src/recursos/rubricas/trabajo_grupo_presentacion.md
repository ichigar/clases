## Rúbrica para la evaluación de trabajo en grupo en 3 fases

### Rúbrica para documento colaborativo en grupo

| **Criterio**                             | **Excelente (9-10 puntos)**                                  | **Bueno (7-8 puntos)**                                       | **Suficiente (5-6 puntos)**                                  | **Insuficiente (<5 puntos)**                                 | **Puntuación** |
| ---------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | -------------- |
| **Participación individual (I)**         | El estudiante aporta a casi todas las preguntas, sus contribuciones están claramente identificadas con su nombre, y cada uno presenta ideas originales o relevantes. | El estudiante contribuyen a todas las preguntas, y sus aportaciones están claramente identificadas con su nombre. | El estudiante no aporta a todas las preguntas o sus contribuciones no están siempre claras. | Pocas o ninguna contribución por parte del estudiante, o no se identifican claramente. |                |
| **Calidad de las respuestas** (I)        | Respuestas bien fundamentadas, con detalles precisos y evidencia clara. Refleja un análisis profundo y comprensión de los temas. | Las respuestas son correctas y muestran un buen nivel de comprensión, pero pueden carecer de profundidad o algunos detalles. | Respuestas correctas pero básicas, con falta de análisis o justificación. | Respuestas incompletas, incorrectas o sin justificación suficiente. |                |
| **Relevancia de las contribuciones** (I) | Las contribuciones de cada estudiante están bien enfocadas en la pregunta, añadiendo valor al trabajo colectivo. | La mayoría de las contribuciones son relevantes y responden a las preguntas de manera adecuada. | Algunas contribuciones son vagas o no se relacionan directamente con la pregunta. | Muchas contribuciones irrelevantes o sin sentido en relación con las preguntas planteadas. |                |

### Rúbrica para evaluar documento con respuestas consensuadas

| **Criterio**                            | **Excelente (9-10 puntos)**                                  | **Bueno (7-8 puntos)**                                       | **Suficiente (5-6 puntos)**                                  | **Insuficiente (<5 puntos)**                                 | **Puntuación** |
| --------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | -------------- |
| **Integración de aportaciones (G)**     | Todas las aportaciones relevantes del documento anterior se han tenido en cuenta y están claramente integradas en la respuesta final de forma coherente y completa. | La mayoría de las aportaciones relevantes han sido integradas de manera adecuada, aunque pueden faltar algunos detalles menores. | Algunas aportaciones importantes no han sido integradas o se presentan de forma superficial. | Muchas aportaciones importantes del documento original han sido omitidas o no integradas de manera coherente. |                |
| **Claridad y cohesión (G)**             | El documento final es claro, bien organizado y fluido. Las ideas están presentadas de manera coherente, sin repeticiones o contradicciones. | El documento es claro en su mayoría, aunque puede tener algunas pequeñas repeticiones o saltos en la cohesión. | El documento es comprensible, pero tiene problemas significativos de organización, repeticiones o falta de cohesión entre las ideas. | El documento es confuso, con ideas mal organizadas o contradictorias, y no sigue un hilo conductor claro. |                |
| **Calidad del análisis y síntesis (G)** | La respuesta final refleja un análisis profundo de las aportaciones previas, sintetizando las ideas de manera eficaz para ofrecer una respuesta completa y bien fundamentada. | Buen nivel de análisis y síntesis, aunque algunas ideas podrían haberse explorado o combinado de manera más efectiva. | La síntesis de las aportaciones es básica o poco profunda, con un análisis limitado de las ideas. | La respuesta final es una simple combinación de las aportaciones sin un análisis o síntesis adecuados. |                |
| **Corrección y presentación**           | La respuesta final está libre de errores gramaticales y ortográficos, y el documento está bien presentado, con un formato claro y profesional. | La respuesta final tiene pocos errores gramaticales o de formato que no afectan la comprensión global. | Hay varios errores gramaticales o de formato que afectan la presentación o la claridad del documento. | Numerosos errores gramaticales, ortográficos o de formato que dificultan la comprensión del documento. |                |

### Rúbrica para Evaluar Exposición con Presentación

| Criterios                           | **Excelente (9-10 puntos)**                                  | **Bueno (7-8 puntos)**                                       | **Suficiente (5-6 puntos)**                                  | **Insuficiente (<5 puntos)**                                 | Puntuación |
| ----------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ---------- |
| **Contenido (G)**                   | Información completa y relevante; se abordan todos los puntos clave. | Información relevante; se abordan la mayoría de los puntos clave. | Información parcial; faltan algunos puntos clave.            | Información irrelevante o muy limitada.                      |            |
| **Estructura (G)**                  | Presentación bien organizada; clara introducción, desarrollo y conclusión. | Presentación organizada; buena estructura, pero con algunos deslices. | Estructura básica; falta claridad en algunas partes.         | Desorganizada; difícil de seguir.                            |            |
| **Uso de la Presentación (G)**      | Diapositivas visualmente atractivas y bien diseñadas; apoyan la exposición. | Diapositivas adecuadas; apoyan la exposición, pero podrían mejorar. | Diapositivas poco atractivas; apoyo limitado a la exposición. | Diapositivas confusas o ausentes.                            |            |
| **Habilidades de Presentación (I)** | Excelente comunicación verbal y no verbal; mantiene el interés del público. | Buena comunicación; mantiene el interés, aunque con algunos nervios. | Comunicación aceptable; a veces pierde el interés del público. | Dificultades significativas en la comunicación; poco interés del público. |            |
| **Dominio del Tema (I)**            | Muestra un dominio excepcional del tema; responde preguntas con confianza. | Muestra buen dominio del tema; responde la mayoría de las preguntas. | Dominio limitado del tema; dificultad para responder preguntas. | No demuestra dominio del tema; no puede responder preguntas. |            |
| **Tiempo de Presentación (I)**      | Cumple con el tiempo asignado de manera efectiva.            | Cumple con el tiempo, pero podría ser más conciso.           | Se excede o no llega al tiempo asignado.                     | Muy desproporcionado en el tiempo.                           |            |
